# <p align="center">Jitsi Meet</p>

Jitsi Meet 是一套开源项目集合，使用户能够使用和部署具有最先进视频质量和功能的视频会议平台。

<hr />

<p align="center">
<img src="https://raw.githubusercontent.com/jitsi/jitsi-meet/master/readme-img1.png" width="900" />
</p>

<hr />

Jitsi Meet 提供的主要功能包括：

* 支持所有主流浏览器
* 移动端应用
* 用于集成的 Web 和原生 SDK
* 高清音视频
* 内容共享
* 举手和表情反应
* 支持私聊的聊天功能
* 投票
* 虚拟背景

以及更多功能！

## 使用 Jitsi Meet

Jitsi Meet 基于浏览器，使用非常简单。访问 [meet.jit.si](https://meet.jit.si) 即可开始体验。它可扩展且免费使用。您只需要一个 Google、Facebook 或 GitHub 账号即可发起会议。支持所有浏览器！

使用手机？没问题，您可以使用手机浏览器或功能完善的移动端应用：

| Android | Android (F-Droid) | iOS |
|:-:|:-:|:-:|
| [<img src="resources/img/google-play-badge.png" height="50">](https://play.google.com/store/apps/details?id=org.jitsi.meet) | [<img src="resources/img/f-droid-badge.png" height="50">](https://f-droid.org/packages/org.jitsi.meet/) | [<img src="resources/img/appstore-badge.png" height="50">](https://itunes.apple.com/us/app/jitsi-meet/id1165103905) |

如果您想抢先体验正在开发中的新功能，也可以注册参加我们的公开测试版：

* [Android](https://play.google.com/apps/testing/org.jitsi.meet)
* [iOS](https://testflight.apple.com/join/isy6ja7S)

## 运行自己的实例

如果您想部署自己的 Jitsi Meet 实例，请前往[使用手册](https://jitsi.github.io/handbook/docs/devops-guide/)开始操作。

我们提供 Debian 软件包和完善的 Docker 部署方案，使部署尽可能简单。
高级用户也可以从源码构建所有组件。

您可以在[这里](https://jitsi.github.io/handbook/docs/releases)查看最新版本。
